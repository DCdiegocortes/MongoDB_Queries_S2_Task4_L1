# 📊 MongoDB Queries – Restaurants Dataset

## 📌 Description  
This project focuses on practicing MongoDB queries using a real dataset of restaurants in New York.

The objective is to retrieve and manipulate data using **MongoDB query syntax**, applying filters, projections, sorting, and operators on a non-relational database structure.

## 🛠 Technologies  
- MongoDB (Compass)  
- JavaScript (MongoDB queries)  

## 📂 Project Structure  

- `restauratnQueries.js` → MongoDB queries implementation  
- `restaurants.json` → Dataset used for testing  
- `README.md` → Project documentation  

##  What was Practiced  

- Basic queries using `find()`  
- Filtering data with operators (`$gt`, `$lt`, `$ne`, `$in`)  
- Projections to control returned fields  
- Working with nested fields and arrays  
- Using `$elemMatch` for array filtering  
- Sorting, limiting, and skipping results  
- Regular expressions (`$regex`) for text search
  
## 🚀 Usage  

1 >> Import `restaurants.json` into MongoDB  
2 >> Open the `restaurants` collection  
3 >> Execute queries from `queries.js`  
4 >> Verify results in MongoDB Compass or the evaluator
